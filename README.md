## 📌 Overview

KrishiSathi is a full-stack intelligent agriculture platform designed to empower farmers with:
- Real-time weather insights  
- AI-based crop disease detection  
- Smart agricultural alerts  
- Digital marketplace ecosystem  

---

## 🚀 Core Features

### 🌤️ Real-Time Weather Monitoring & Alerts
- Live weather updates via OpenWeather API  
- Backend-controlled architecture using Flask  
- Auto-refresh every 60 seconds  

### 🌿 Crop Disease Prediction
- Upload crop leaf images  
- CNN model detects disease  
- Outputs disease, confidence, treatment  

### 🌱 Crop Recommendation *(Planned)*
- Based on soil, pH, moisture, season  

### 🛒 Marketplace *(Planned)*
- Buy/Sell seeds, fertilizers, tools

npm run dev
