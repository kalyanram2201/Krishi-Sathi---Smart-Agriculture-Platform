# KrishiSathi 🌾

**KrishiSathi** is a smart agriculture web platform designed to help farmers make better decisions using modern technologies like AI, weather analytics, and digital marketplaces.

The platform provides crop disease detection, crop suggestions, real-time weather forecasts, and an integrated agri marketplace — all in one place.

---

## 🚀 Features

### 🌱 AI Crop Disease Detection
- Upload crop leaf images
- Detect diseases (demo-based AI logic)
- Get organic & chemical treatment suggestions

### 🌾 Smart Crop Suggestions
- Recommendations based on:
  - Soil type
  - pH value
  - Moisture level
  - Season & location
- Profitability and yield insights

### 🌦️ Weather Forecast & Alerts
- Current weather conditions
- 5-day forecast
- Farm-specific alerts
- Uses OpenWeather API (with demo fallback)

### 🛒 Agri Marketplace
- Browse seeds, fertilizers, tools & equipment
- Filter by category
- Sort by price, rating, popularity

### 🛍️ Cart System
- Add/remove items
- Quantity control
- Checkout flow (demo)

---

## 🧠 Tech Stack

- React (JavaScript)
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- Axios

---

## 🔐 Environment Variables

Create a `.env` file:

VITE_OPENWEATHER_API_KEY=YOUR_OPENWEATHER_API_KEY  
VITE_RAZORPAY_KEY_ID=YOUR_RAZORPAY_KEY
